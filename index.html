nibannme OS X HP

概要

nibannme OS X は、Batch と Python を組み合わせて構築された OS 風の Live 環境です。 起動画面、GRUB 風メニュー、Live DOS、インストーラーなどを備え、 実際の OS のような体験ができるプロジェクトです。

特徴

  • Batch ベースの起動画面
  • Python による GRUB 風ブートメニュー
  • Live DOS 環境での操作
  • インストーラー風 UI
  • 日本語対応
  • 軽量でシンプルな構成
